# BrightLearn-Data-Analytics-Research-Assignment-1 QUESTIONS
This question paper forms part of Research Assignment 1 for the Data Analytics & Data Science course offered by BrightLearn. It is designed to evaluate a beginner’s understanding of essential concepts in data analytics, artificial intelligence, and modern data technologies.

The paper is structured into five sections, each focusing on a specific area of learning. Section A: Data Fundamentals introduces core concepts such as databases, data warehouses, data lakes, data modeling, ETL processes, and data storage formats. This section builds a foundation for understanding how data is stored, organized, and processed.

Section B: Artificial Intelligence & Machine Learning explores key ideas in AI, including machine learning types (supervised, unsupervised, and reinforcement learning), deep learning, neural networks, natural language processing (NLP), and generative AI. It helps learners understand how intelligent systems work and how they are applied in real-world scenarios.

Section C: APIs, Tools & Developer Concepts focuses on technical skills and tools used in data and software development. Topics include APIs, REST APIs, JSON, cloud computing, version control systems like Git, Jupyter Notebooks, and the Python programming language.

Section D: Data Ethics, Governance & Careers highlights the importance of responsible data use. It covers data privacy, data ethics, bias in AI, data governance, and different career roles such as data analyst, data scientist, and data engineer.

Finally, Section E: Personal Reflection encourages learners to reflect on their learning experience, challenges faced, and how data and AI may influence their future careers.

Overall, the question paper aims to develop both theoretical knowledge and practical awareness, while promoting critical thinking and ethical responsibility in the field of data analytics.
